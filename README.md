# exploratory-data-analysis
A full data cleaning and exploratory analysis project using Python to uncover insights from global sales data across products, countries, and regions.

Project Description

This project focuses on cleaning and analyzing a global sales dataset using Python.
The data includes multi-year sales records, product categories, and country/region information from three tables: events, products, and countries.

Main Steps  
Data loading and exploratory overview of table structures  
Data cleaning: missing values, incorrect types, duplicates, anomalies  
Merging all tables into a single analytical dataset  
Analysis of sales by product categories, countries, and regions  
Analysis of shipping time and its impact on revenue  
Time-series exploration: sales trends over months and years  
Weekly patterns and seasonality analysis  
Final report in Google Colab with visualizations and business insights

Tools  
Python (pandas, numpy, matplotlib, seaborn) in Google Colab.
